Local errors demo

This page gives an example of global embed errors. The first embed below has a src that does not match the origin of the page (http://localhost:8081, by default). This error displays in the first embed, and carries over to the one beneath it.

This embed has two errors. The first is global: it, too, does not match the page's origin. The second is local to this embed only; its src is for a deployment of CB other than the one specified by the src of the first embed on the page. All embeds in a single page must be from one CB deployment.
